## Technical aspects

The implementation of this application uses SwiftUI.

## Usage

Clone this repository and open `Zonda-Tickers-iOS.xcodeproj`.

```bash
$ git clone https://github.com/albinekcom/Zonda-Ticker-iOS.git
$ cd Zonda-Ticker-iOS
$ open Zonda-Ticker-iOS.xcodeproj
```
